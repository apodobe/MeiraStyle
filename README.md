# MeiraStyle / osipovastyle.ru

Монорепозиторий: **сайт** (Astro), **стратегия бренда** (автономный HTML), **канвас Cursor**, **документы** (метод, план Telegram).

## Структура

| Путь | Назначение |
|------|------------|
| `сайт/` | Проект Astro: `npm install` и `npm run dev` / `build` **из этой папки**. Исходники, `public/`, `scripts/`, `deploy/`, `photos/`. |
| `стратегия-бренда/` | Файл `бренд-стратегия.html` — полная стратегия для печати и `file://`; инструкция в `ЧИТАЙМЕ.md`. |
| `документы/` | `метод-свой-знак-полный-документ.md`, `телеграм-приMEIRAчная-30-дней.md` и др. |
| `canvases/` | **Не переносить и не переименовывать** без проверки в Cursor: путь `canvases/*.canvas.tsx` используется IDE для Canvas. |

## Переменные окружения

Файл `.env` в **корне репозитория** (рядом с `сайт/`) — основной для FTP, Telegram и локального API бронирования. Скрипты в `сайт/scripts/` подхватывают сначала корневой `.env`, при необходимости — `сайт/.env`.

## Команды (сайт)

```bash
cd сайт
npm install
npm run dev
```

Секреты не коммитить: в корне есть `.gitignore` для `.env`, `node_modules`, артефактов сборки.
