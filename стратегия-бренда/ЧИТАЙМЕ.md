# Метод «Свой знак» — автономная стратегия (без Cursor)

Папка для открытия на любом Mac **без установки Cursor**.

## Как открыть

1. Скопируйте всю папку `стратегия-бренда` на другой компьютер (AirDrop, облако, USB).
2. Дважды щёлкните по файлу **`бренд-стратегия.html`** — откроется в **Safari** или **Chrome** (или перетащите файл в окно браузера).

Работает по протоколу `file://` — сервер не нужен.

## Содержимое

| Файл | Назначение |
|------|------------|
| `бренд-стратегия.html` | Полная стратегия: вкладки Стратегия, Метод, Контент, TODO, Медиа, Риски (то же содержание, что расширенный канвас в Cursor). |
| `ЧИТАЙМЕ.md` | Эта инструкция. |

## Связь с проектом в Cursor

Интерактивный канвас в IDE: `canvases/brand-strategy.canvas.tsx` — при изменении стратегии имеет смысл обновить и `бренд-стратегия.html`, чтобы копии на других машинах оставались актуальными.
